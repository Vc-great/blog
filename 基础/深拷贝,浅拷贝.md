# 拷贝

## 赋值

```js
let a = {a:1}
let b = a
```

新变量的的指针指向原实例的同一个地址
`特点:`会相互影响

## 浅拷贝

​	基本数据类型相互独立;

​	引用类型指向同一地址,会相互影响
`特点:`引用类型会相互影响

![img](D:\个人\Blog\基础\Untitled.assets\16ce894a1f1b5c32)

### 实现

```js
function clone(target){
	let cloneTarget = {};
	for(const key in target){
		cloneTarget[key] = target[key];
	}
	return cloneTarget;
}
```

## 深拷贝

​	完全不同的两个实例

​	`特点:`变量指向的内容一样但是相互独立,互不影响
